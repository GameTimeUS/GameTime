What is a tweak/theme?
A tweak is a modification that you can download and apply to a jailbroken cellphone specifically IOS
A theme is a certain kind of tweak. Themes change the way IOS looks including UI and App icons.

What is a Jailbreak?
A jailbreak is a utility that can be used to break restrictions to IOS. A jailbreak usually involves getting access to the file system 
of IOS, developers usually achieve this by an exploit. Once the jailbreak is achieved, developers usually intergrate the jailbreak into an app which he releases and all you have to do is press JAILBREAK and the app hacks into your phone and gives you access to every file on your phone including system files and gives you the ability to install apps and modifications Apple usually wouldn't allow you to install. The Jailbreak Community made tweak installations very easy by creating cydia, an alternative app store that you get with the jailbreak so its easy to install tweaks.
