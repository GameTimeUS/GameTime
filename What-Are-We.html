Heres a little bit about me!

My name is Noah Wolcott, I am a senior in high school. I will be attending East Tennessee State University in the fall to earn my major degree in Computer Science/Computer Technologies. I have been obsessed with computers since I was 6 years old and taught my self everything I know. starting from learning Program languages including, Python, C#, .NET UI, and HTML. When I turned 15 I wanted to try something new, i wanted to build a video game that at first was going to be kept to myself privately, but later I became so proud of how much work I did on it, that I had to let the world see. Although nobody really downloaded it, I posted the game on drop box and sent the link out to everyone I knew. the Game was not popular at all but I had fun with it so I continued to push updates out on the game including small patches and big feature updates. after a while of me not getting anywhere I stopped updating it. I gave up. I guess I got bored. Well along the way I found the Project in the cloud storage I forgot I had, granted it was a way older build. after creating the game I decided I wanted to make a name for the development group... Hint: just me for now. I decided to call it Game Time. when I was younger I loved the idea of having my own "company name" it made me feel like a pro developer, so i kept it i made a logo, and hey look i even managed to code my own website.
Anyway thanks for visiting my site, and looking at my projects! The original game is the only available game currently but I plan to have more software and games very soon. currently I'm working on a launcher/client to run the game which should be available some time in June.  

What is Game Time

Game Time a game developer. Although only one person is apart the organization, Noah thought it would look more professional if he had a brand name for his game. So that's what this is although hopefully in the near future it will be a group of developers and not just me. I am hoping that Game Time will push more than one thing out and soon so this is not a name for one developer with one game I will have multiple pieces of software and games to show everryone.
